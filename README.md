# Game2D-BraveLink-Qt
Development of a 2D Game according to the MVC model in C ++

You can use Numeric keyboard for move your Hero princess Link  or use just your Arrow
4 : Left 
5 : Down 
6 : Right
8 : Up 
p : pause

If you take a weapon same as  scissors or extinguisher :
x : scissors
w : extinguisher
